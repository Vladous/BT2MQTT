# \# BT2MQTT

# 

# BT2MQTT is an Android-only .NET 10 MAUI application that bridges Bluetooth devices to MQTT.

# 

# \## Purpose

# 

# The application is designed to provide a reliable communication bridge between Bluetooth devices and MQTT infrastructure without requiring custom hardware modems or unstable third-party bridge applications.

# 

# \## Features

# 

# \- Android-only implementation

# \- Built with .NET 10 and .NET MAUI

# \- Bluetooth Low Energy support

# \- MQTT publish and subscribe

# \- Persistent application settings

# \- Automatic reconnect

# \- Planned foreground service support for long-running operation

# 

# \## Technology Stack

# 

# \- .NET 10

# \- .NET MAUI

# \- Plugin.BLE

# \- MQTTnet

# 

# \## Project Status

#

# \## Roadmap

#

# \- Settings persistence

# \- Automatic reconnect

# \- Foreground service support
